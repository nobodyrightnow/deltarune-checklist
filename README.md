# Deltarune Completionist Checklist
## Overview
This website covers all of the missable content in Deltarune. 

No user data is stored.

This project was built as a personal learning exercise in web development.

## Features
- Track consumables, weapons, armor, key items, choices, and titles
- Route-specific filtering (Pacifist, No-Mercy, Weird Route)
- Option to include bonus content in the checklist (cutscenes, character development, etc.)
- Persistent progress using localStorage
- Reset button to clear progress and localStorage

## How to Use
1. Open the website at https://nobodyrightnow.github.io/deltarune-checklist/ 
2. Select route
3. Play Deltarune and check things off as you go
4. Come back at any time to where you left off with persistent storage
